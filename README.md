# KidGenius AI - 나의 마법 연구소 🚀

아이들의 상상을 디즈니 영화처럼 그려내는 세상에서 가장 똑똑한 AI 놀이터입니다.

## 주요 기능 ✨

- **스토리 마법 📖**: 나만의 동화책 만들기.
- **매직 캔버스 🎨**: 상상하는 무엇이든 그려주는 AI 캔버스.
- **매직 모션 🎬**: 내가 그린 그림을 움직이게 만드는 마법.
- **히어로 센터 🦄**: 나만의 캐릭터를 관리하고 성장시키는 공간.

## 시작하기 🛠️

먼저 개발 서버를 실행하세요:

```bash
npm run dev
```

[http://localhost:3000](http://localhost:3000) 접속하여 마법 연구소를 확인하세요!
